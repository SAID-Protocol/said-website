Quick Start

Get your agent on-chain in under 5 minutes.

Step 1

Install dependencies and clone the SDK

Terminal 
# Clone the repo
git clone https://github.com/kaiclawd/said.git
cd said

# Install Anchor dependencies
yarn install
Step 2

Register your agent identity (free — you only pay Solana rent ~0.002 SOL)

TypeScript 
import { Program } from "@coral-xyz/anchor";
import { PublicKey } from "@solana/web3.js";

// Program ID (devnet)
const SAID_PROGRAM_ID = new PublicKey(
  "5dpw6KEQPn248pnkkaYyWfHwu2nfb3LUMbTucb6LaA8G"
);

// Your agent's metadata (host this JSON somewhere)
const metadataUri = "https://yoursite.com/agent.json";

// Derive your agent's PDA
const [agentPDA] = PublicKey.findProgramAddressSync(
  [Buffer.from("agent"), wallet.publicKey.toBuffer()],
  SAID_PROGRAM_ID
);

// Register (free!)
await program.methods
  .registerAgent(metadataUri)
  .accounts({
    agentIdentity: agentPDA,
    owner: wallet.publicKey,
    systemProgram: SystemProgram.programId,
  })
  .rpc();

console.log("✅ Agent registered:", agentPDA.toString());
Step 3 (Optional)

Get verified for 0.01 SOL — adds a verified badge to your identity

TypeScript 
// Treasury PDA (receives verification fees)
const [treasuryPDA] = PublicKey.findProgramAddressSync(
  [Buffer.from("treasury")],
  SAID_PROGRAM_ID
);

// Get verified (0.01 SOL)
await program.methods
  .getVerified()
  .accounts({
    agentIdentity: agentPDA,
    treasury: treasuryPDA,
    owner: wallet.publicKey,
    systemProgram: SystemProgram.programId,
  })
  .rpc();

console.log("✅ Agent verified!");

How it works

Three simple steps to give your agent a verifiable on-chain identity.

1

Register

Call registerAgent() with your metadata URI. Free. Creates a unique PDA for your agent.

2

Verify

Pay 0.01 SOL to get a verified badge. Stand out from unverified agents.

3

Build Reputation

Collect feedback from interactions. On-chain reputation score updates in real-time.
